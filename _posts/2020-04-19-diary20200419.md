# 자동 정렬
crtl+all 로 전체 블록 후 crtl + K + F

# 블록 주석처리 / 해제
주석 - crtl + K + C / 해제 - crtl + K + U

# 한줄 삭제 
crtl + L 
# 바로 윗줄 복사
crtl + D

# 코드 움직이기 
alt + 위,아래키 

# 무조건 엔터 
crtl + shift + 엔터 ( 자동완성으로 뒤에 ) } 이런거 무시하고 글 중간이 아니라 바로 엔터가 쳐짐)


# 다중 포인터 만들기 
예를들면 

```
a|
b|
c|
```

이렇게 다중 포인터 사용 가능. crtl + shift + 클릭

# 다중 변환 
crtl + H ( crtl + f 는 찾는거면, 이건 바꾸는거 ) 

# 정의 부분 바로 가기 
crtl + 클릭  ( 그외 커서 돌아가기 crtl '-' )
