일단은 사용중입니다. 

테마 바꾸면 바꿔야 할 것도 많고, 

주소 체계가 바뀌면 한동안 구글과 네이버에 등록 해 놓은 

검색이 엉망이 되어버릴 것이기 때문에

바꾸고 싶은 부분은 직접 바꾸는 것으로 가지 않을까 싶습니다.
